FAQ

Common questions

What exactly is MiroFish and how does it differ from ChatGPT? +

MiroFish is not a single-model chatbot — it's a multi-agent simulation engine. Where ChatGPT generates a single AI response, MiroFish spawns thousands of independent AI agents representing different personas, lets them interact organically, and synthesizes the emergent collective behavior into a structured prediction report. You get a simulated society, not just a chatbot answer.

What kind of seed data does MiroFish accept? +

MiroFish accepts plain text input (typed directly), as well as uploaded files in PDF, Markdown (.md), and TXT formats. Seed content can be news articles, policy drafts, financial reports, research papers, or even novel chapters. The engine extracts entities and relationships automatically via GraphRAG.

Which LLM providers does MiroFish support? +

MiroFish supports any LLM API that follows the OpenAI SDK format — including OpenAI GPT-4, Alibaba Qwen, Anthropic Claude, and local models via compatible proxies. The recommended model for balanced performance and cost is Alibaba Qwen-plus via the Bailian Platform. Cloud plans include managed LLM access; self-hosted plans require your own API key.

How long does a simulation take to run? +

Simulation time depends on agent count and round count. A 20-agent, 20-round simulation typically completes in 5–10 minutes. Larger simulations (500+ agents, 80+ rounds) can take 30–60 minutes. Pro and Enterprise plans benefit from parallel processing infrastructure that significantly reduces wall-clock time.

Can I self-host MiroFish? +

Yes. MiroFish is fully open source under the AGPL-3.0 license. You can deploy it yourself using source code (Node.js 18+ frontend, Python 3.11–3.12 backend) or via Docker Compose with a single command. Self-hosted deployments have no agent or simulation limits beyond your own compute resources.
